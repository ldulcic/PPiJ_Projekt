Dijkstra's algorithm


What is it?

Dijkstra's algorithm is an algorithm that finds shortest path between two nodes in a graph.

It was conceived in 1956. by computer scientist Edsger W. Dijkstra


Why does it matter?

Most notably, Dijkstra's algorithm is used in networking. Services that we use daily, like the Internet, could not work without this algorithm.
