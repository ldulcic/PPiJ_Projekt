Dijkstra's algorithm


What is it?

Dijkstra's algorithm is an algorithm that finds shortest path between two nodes in a graph.

It was conceived in 1956. by computer scientist Edsger W. Dijkstra


Why does it matter?

Most notably, Dijkstra's algorithm is used in networking. Services that we use daily, like the Internet, could not work without this algorithm.

Breadth-first search


What is it?

Breadth first search algorithm is an algorithm that traverses a graph level by level.

It was invented by E.F.Moore in late and 1950s.


Why does it matter?

Breadth first search has many applications in IT and beyond, most notably in GPS navigation systems and P2P networks like BitTorrent.
